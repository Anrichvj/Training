Here is photo examples of my projects...

April Ford is a website page that I built using HTML and CSS, utilizing divs and flexbox to create a clean and modern design. As my first ever website project, I learned a lot about the basics of web development and design principles. By using divs to structure the content and flexbox to organize the layout, I was able to create an intuitive and visually appealing user interface. Through this project, I developed a greater appreciation for the importance of responsive design and user experience, and I look forward to applying these skills in future web development projects.

Minimalist is a website page that I created using HTML and CSS. The design is based on the minimalist art movement, using a clean and simple layout with limited color and typography. By utilizing various CSS techniques such as grid, flexbox, and responsive design, I was able to create a sleek and modern look while maintaining the minimalist aesthetic. Through this project, I learned how to balance functionality and design to create an effective user interface. The project taught me the importance of attention to detail and how small changes can make a big impact on the overall design

The EOH Login Page is a project I created using HTML and CSS. The design is focused on creating an easy-to-use login experience for users. I used CSS to create a clean and modern layout that is both visually appealing and functional. The page includes various form fields for the user to input their login credentials, with CSS styling to make it clear what information is required.Through this project, I learned how to create a responsive and user-friendly login page that meets the needs of both the user and the business
